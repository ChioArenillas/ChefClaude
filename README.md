# Chef Claude

## 🧩 Description

Chef Claude is a React-based web application that allows users to interact with an intelligent cooking assistant to generate recipes based on selected ingredients.

The project focuses on user input handling, dynamic content generation, and creating an intuitive and engaging user experience.

## 🚀 Live Demo

https://chef-claude-bice.vercel.app/

## 🚀 Technologies

* React
* JavaScript (ES6+)
* CSS
* Vercel (deployment)

## 🎯 Features

* Ingredient selection and management
* Dynamic recipe generation
* Interactive user input
* Conditional rendering based on user actions
* Clean and responsive UI

## 🧠 Key Concepts

* State management with React hooks
* Handling user input and form logic
* Conditional rendering
* Component-based architecture
* Separation of logic and presentation

## 💡 Purpose

This project was built to simulate a smart cooking assistant and explore how dynamic data can be generated based on user input.
